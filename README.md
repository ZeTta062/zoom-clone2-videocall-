# Zoom!
Zoom Clone using NodeJS, WebRTC and Websockets.

## 구현 기능
- 닉네임 저장기능 (미설정 시 Anonymous로 나오게 구현)
- 채팅방 이름 설정
- 공개방이 있을 시 공개방 목록을 만듦
- 채팅방에 다른사람이 들어왔을 시 알림
- 채팅방에서 다른사람이 퇴장하였을 시 알림
- 채팅 기능 구현(다른 사람은 닉네임으로, 본인의 채팅은 You로 닉네임 표시)

## 완성작품
미완성
